# Decision Playbook Website

This repository contains a minimal interactive website located in the `web/` folder. Open `index.html` in a browser to try it.

The page lets you:

- Read a short summary of the playbook.
- Add, remove, or complete daily action items (stored in your browser).
- Reset to a set of default tasks inspired by the playbook.
- See a progress bar showing completion percentage.

The site is intentionally simple so it can run without a server. Open `web/index.html` in any modern browser to get started. Your tasks are saved locally, so no network is required.
